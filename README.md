## 3. 핵심 기능
이 서비스의 핵심 기능은 공공데이터포털에서 제공하는 국내 숙박업소정보api를 활용하여 제작하였으며 숙소 정보를 볼 수 있으며 날짜와 인원수를 선택하여 예약을 할 수 있는 기능을 제공합니다.
